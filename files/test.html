Preliminaries

Denote by I+ the unit interval with a disjoint basepoint and by I the unit interval based at 1. The interval I is a cylinder objects that detects left homotopies in the category of orthogonal spectra. Such a homotopy f ≃ g is given by a commutative diagram:

$$\begin{tikzcd} X \\ & {X\wedge I_+ } & Y \\ X \arrow["{i_1}"', from=1-1, to=2-2] \arrow[from=2-2, to=2-3] \arrow["f", from=1-1, to=2-3] \arrow["{i_0}", from=3-1, to=2-2] \arrow["g"', from=3-1, to=2-3] \end{tikzcd}$$

Given maps f1, f2 : X → Z and g1, g2 : Y → W with homotopies Hf between f1 ≃ f2 and Hg between g1 ≃ g2 we get a third homotopy: H∧ : X ∧ Y → Z ∧ W given by: $$X\wedge Y\wedge I_+ \to X\wedge Y \wedge I_+^2\xrightarrow{H_1\wedge H_2} Z\wedge W$$ Where the first map is induced by the infinite suspension of the diagonal on I+. We will denote the smash homotopy H∧ by H1∧≃H2.

We may identify In with the unit ball in ℝn. Allow ∂ : Sn − 1 → In denote the inclusion of the boundary. For all of these maps and spaces, we will use the same notation for their infinite suspensions. If X is a spectrum, denote X ∧ I by CX. Let 𝕊 denote the sphere spectrum Σ∞S0. Then there is always a map X → CX given by: $$\iota: X\xrightarrow{\cong}X\wedge\mathbb{S}\xrightarrow{\text{id}\wedge\partial}CX$$ We will say that a map f : CX → Y restricts to a map g : X → Y if f ∘ ι = g.

Lemma 1. A map of spectra f : X → Y is nullhomotopic if and only if there exists a map g : CX → Y that restricts to f.

Homotopical and Homological Massey Products

Spaces of the form (⋀i < k < j(C(Σnk𝕊)) ∧ Σnj𝕊 will play an important role in our construction. After swapping smash terms, we may indentify these spaces with (⋀i < k ≤ jΣnk𝕊) ∧ Ij − i − 1. Spaces of the form Y ∧ Im recieve maps ∂ from Y ∧ Σ∞Sm − 1 ≅ Σm − 1Y induced by the space-level boundary inclusion of a sphere into the disk In. Similarly to the above, we will say that a map f : Y ∧ Im → Z restricts to a map g : Σm − 1Y → Z if ∂f := f ∘ ∂ = g.

Let X be an E∞-ring spectrum. Suppose we are given homotopy classes ai ∈ πniX for 1 ≤ i ≤ m.

Definition 2 (Defining Systems). A defining system for  < α1, ..., αm> is given by maps {Ai − 1, i : Σni𝕊 → X}1 ≤ i ≤ m. Such that Ai − 1, i represents ai as well as maps: Ai, j : (⋀kC(Σnk𝕊)) ∧ Σnj𝕊 → X For 1 ≤ i < j ≤ m, (i,j) ≠ (0,m) and i < k < j, satisfying the following condition (*). We will say that Ai, j has level j − i.

Definition 3 (Coherence condition (*)). Given Ai, j as above there are maps Ai, k, j := μX(Ai, k∧Ak, j). The domain of Ai, k, j is of the form: (⋀i < ℓ < kC(Σnℓ𝕊)) ∧ Σnk𝕊 ∧ (⋀k < ℓ < jC(Σnℓ𝕊)) ∧ Σnj𝕊 Let Yk ∧ Ij − i − 1 denote dom (Ai, k, j) after swapping terms and denote by ιk the obvious map ιk : ⋀i < ℓ ≤ jΣnℓ𝕊 → Yk. Then: colimkιk = Σj − i − 2⋀i < ℓ ≤ jΣnℓ𝕊 Because the maps Ai, k, j ∘ ιk agree, they induce a map Ãi, j : colimkιk → X. Then condition (*) is that ∂Ai, j = Ãi, j.

The condition (*) makes it clear that the construction of a level n element of the defining system depends heavily but also only on the choices of level k < n elements. One can therefore think of building a defining system in stages, one for each level, in a pyramid fashion.

Definition 4 (Massey Product). The Massey product  < a1, ..., am> is defined if there exists a defining system Ai, j. Note that while A0, m is not defined, we may still construct Ã0, m. We say that the homotopy class of Ã0, m is in  < a1, ..., am> and that the latter set is the collection of all such induced maps from defining systems.

Let E, F be E∞-ring spectra in a symmetric monoidal model category for spectra. Then E, F are symmetric monoids in this category with products μE, μF respectively. If a : Σn1𝕊 → E ∧ F and b : Σn2𝕊 → E ∧ F are two homology classes in E*F, their product is defined by: $$ab:\Sigma^{n_1+n_2}\mathbb{S}\xrightarrow{\cong}\Sigma^{n_1}\mathbb{S}\wedge\Sigma^{n_2}\mathbb{S}\xrightarrow{a\wedge b}E\wedge F\wedge E\wedge F \xrightarrow{\cong}E^2\wedge F^2\xrightarrow{\mu_E\wedge\mu_F}E\wedge F$$ Indeed, E ∧ F is again a commutative monoid by a general categorical argument and as a result, the above construction defines Massey products in E*F.

Lemma 5. If  < a1, ..., am> is defined then any element of the Massey product can be obtained from a fixed choice of level 1 representatives Ai − 1, i for the ai.

Proof. Let Bi, j be complete defining system. We may by assumption choose homotopies Hi − 1, i from Bi − 1, i to Ai − 1, i. Suppose we have defined maps Ai, j up to level n < m − 1 satisfying condition (*) and equipped with homotopies Hi, j from Bi, j to Ai, j. We inductively define Ai′, j′ for j′ − i′ = n + 1. Note that we may already form the maps Ai′, k, j′ as they come from smashes of lower-level maps. Moreover, the smash homotopies of the Hi, j from lower levels (Definition [todo]) yield $B_{i',k,j'}\xrightarrow{\simeq}A_{i',k,j'}$. Therefore the colimit-induced maps $\tilde B_{i',j'}\xrightarrow{\simeq}\tilde A_{i',j'}$ are also homotopic. From Lemma [todo] the map Bi′, j′ is equivalent to the data of a nullhomotopy for B̃, i′, j′. Vertical composition of homotopies yields Ai′, j′ and clearly these two nullhomotopies are themselves homotopic.

Once the system is complete, by the same smashing homotopy argument the maps B̃0, m ≃ Ã0, m are homotopic. ◻

Definition 6. A defined Massey product  < a1, ..., am> is said to be strictly defined if for all 1 ≤ i < j ≤ m, (i,j) ≠ (1,m) the products  < ai, ..., aj> contain only 0.

Lemma 7. A Massey product  < a1, ..., am> is strictly defined if and only if every partial defining system completed to level k < m − 1 can be completed to a full defining system.

Proof. If the system Ai, j is completed up to level k, defining Ai′, j′ for j′ − i′ = k + 1 is equivalent to being able to choose a nullhomotopy for each Ãi, j, j − i = k. This is possible if and only if every possible Ãi, j is nullhomotopic. ◻
